# HR Analytics Dashboard (Advanced Excel)

## Project Overview

This project is an interactive HR Analytics Dashboard built using Microsoft Excel to analyze employee attrition, workforce demographics, and employee performance metrics.

The dashboard helps HR teams and management identify key factors contributing to employee turnover and make data-driven workforce decisions.

## Objective

To monitor employee attrition trends and provide insights into:

- Employee attrition rate
- Department-wise attrition
- Attrition by age group
- Attrition by education level
- Attrition by job role
- Gender distribution
- Marital status analysis
- Workforce demographics
